## 程序简介
程序对接了 Jbox API 和 WebDAV 协议，用户可以通过 WebDAV 协议访问 Jbox，借助 Raidrive 可将 Jbox 挂载为网络磁盘，与资源管理器深度整合，使用体验接近本地磁盘。
