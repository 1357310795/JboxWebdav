# 安装步骤

  1. 根据微软官方文档 (opens new window)安装.NET 6.0 Runtime

提示

只需安装 Runtime，无需安装 SDK

省流：

# Ubuntu
wget https://packages.microsoft.com/config/ubuntu/22.04/packages-microsoft-prod.deb -O packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
sudo apt-get update
sudo apt-get install -y dotnet-runtime-6.0

  1. 在Releases (opens new window)下载JboxWebdav的可执行文件（请选择以 ConsoleApp-Linux 结尾的项目）。
  2. 解压，在文件夹内启动终端
  3. 使用命令JboxWebdav.ConsoleApp运行程序。
  4. 根据提示登录后，服务自动运行。

  1. 接下来请阅读验证可用性&下一步

# 验证可用性&下一步

打开浏览器访问您在 JboxWebdav 里设置的监听地址（默认为 http://127.0.0.1:65472/ (opens new window) ），若看到JboxWebdav Server Running!则表示服务运行成功。
