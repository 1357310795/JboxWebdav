# 在 Windows 系统下安装

# 可用的应用程序类型

JboxWebdav 提供多种应用程序框架，以达到跨平台支持。以下是在 Windows 系统下可用的应用程序框架：

  • WPF 框架（推荐）
  • MAUI 框架
  • 控制台应用

# 使用 WPF 框架

# 安装步骤

  1. 点击这里 (opens new window)下载安装.NET 6.0运行时框架。点击 “运行桌面应用” 下的“下载x64”按钮以下载安装包。下载完成后运行，根据向导进行安装。

提示

如果你知道什么是.NET 6.0运行时，并且明确计算机中已经安装，可以跳过这一步。

  1. 在Releases (opens new window)下载 JboxWebdav 的可执行文件（请选择包含 WpfApp-Windows 的条目）。
  2. 直接运行程序。
  3. 若未处于交大内网环境，程序会提示连接交大 VPN，连接好后继续。

  1. 登录 jAccount（安全性问题见这里）

  1. 直接点击“启动”按钮运行 WebDAV 服务。

  1. 最小化程序，可以让程序在后台运行。

  2. 接下来请阅读验证可用性&下一步

# 使用 MAUI 框架

# 安装步骤

  1. 在Releases (opens new window)下载 JboxWebdav 的可执行文件（请选择包含 MauiApp-Windows 的条目）。
  2. 解压压缩包

  1. 在Install.ps1上右键，点击使用 PowerShell 运行，根据提示安装证书和应用。
  2. 在开始菜单找到 JboxWebdav，点击启动应用。
  3. 若未处于交大内网环境，程序会提示连接交大 VPN，连接好后继续。

  1. 登录 jAccount（安全性问题见这里）

  1. 直接点击“运行”按钮运行 WebDAV 服务。

  1. 接下来请阅读验证可用性&下一步

# 使用控制台应用

# 安装步骤

  1. 点击这里 (opens new window)下载安装.NET 6.0运行时框架。点击 “运行控制台应用” 下的“下载x64”按钮以下载安装包。下载完成后运行，根据向导进行安装。

提示

如果你知道什么是.NET 6.0运行时，并且明确计算机中已经安装，可以跳过这一步。

  1. 在Releases (opens new window)下载 JboxWebdav 的可执行文件（请选择包含 ConsoleApp-Windows 的条目）。
  2. 直接运行程序。
  3. 根据提示登录后，服务自动运行。

  1. 接下来请阅读验证可用性&下一步
