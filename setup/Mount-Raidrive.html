# 使用 Raidrive 将 WebDAV 服务器映射为网络磁盘推荐

注意

Raidrive 仅支持 Windows 系统

# 安装步骤

  1. 打开 Softpedia (opens new window) 下载站，点击Softpedia Secure Download (US) - x64下载 64 位安装包。
  2. 下载完成后安装应用

  1. 安装完成，打开 Raidrive，点击标题栏上的添加。
  2. 找到 NAS 选项卡下的 WebDAV

  1. 按图中进行设置。

特别提醒

确保地址以 http 开头而不是 https，JboxWebdav 不支持 https

  1. 点击确定，然后点击连接。

  2. 在“此电脑”可以看到由 jBox 映射的网络磁盘。
